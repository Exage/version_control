# Основные команды Git

### Инициализация репозитория
Чтобы создать новый локальный репозиторий:
```bash
git init
```

Скопировать удалённый репозиторий:
```bash
git clone <url>
```

Добавить файлы из репозитория
```bash
git add .
```

Закрепить изменения
```bash
git commit -m "ваше сообщение"
```
